## Description

Тестовое задание в компанию `infotecs` на позицию Стажер JS-Разработчик

Приложение отображает таблицу с информацией о пользователях. Вся информация берется через fetch запрос на `DummyJSON /users`.

**Используемый стек:**

- React
- JS
- Tailwind CSS

### Features
- Поддерживает сортировку по возрастанию и убыванию
- Поддерживает фильтрацию по тем же полям, по которым доступна сортировка
- Реализована пагинация

### Possible Improvements
- Обработка ошибок
- Адаптация под мобильные устройства
- Отображение модального окна
